## 🌟 Overview
Building a smart Telegram bot has never been easier. This production-ready boilerplate comes pre-configured with Telegraf and the official OpenAI SDK. It handles long-polling, webhook setups, and basic conversational state management out of the box.

## ✨ Key Features
- 🧠 **OpenAI Integrated**: ChatGPT API ready out-of-the-box.
- 🏗️ **Robust Architecture**: Built with modern Telegraf framework best practices.
- ⚡ **Environment Ready**: Pre-configured `.env` loading and error handling.

## 🚀 Getting Started

### Prerequisites
Make sure you have `Node.js` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/telegram-ai-bot-starter.git
cd telegram-ai-bot-starter
npm install
```

### Usage
```bash
npm start
```
